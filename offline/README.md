# CreativeReporter — 오프라인 완전판 (인터넷 없이 100% 동일 작동)

인터넷이 완전히 차단된 환경에서 시연할 수 있도록, Supabase에 있던 **모든 데이터**와
소재 **미리보기 이미지·영상**을 전부 로컬 파일로 내려받아 담은 오프라인 패키지입니다.
네트워크 요청을 **단 한 건도** 하지 않고 온라인과 똑같이 동작합니다. (헤드리스 브라우저로
네트워크 전면 차단 상태에서 검증 완료 — 외부 요청 0건, 콘솔 에러 0건)

---

## 🚀 사용법 (제일 쉬운 방법)

1. 압축을 풀어 폴더를 통째로 시연 노트북에 옮깁니다. (USB 등)
2. 폴더 안의 **`index.html`** 을 브라우저(크롬 권장)로 엽니다. (더블클릭)
3. 첫 화면에서 **평소 쓰던 비밀번호**를 입력하면 끝. 온라인과 완전히 동일합니다.
   - 보기 전용으로 열려면 뷰어 비밀번호 **`0715`** 를 입력하세요.

> 인터넷이 필요 없습니다. 비행기 모드에서도, 랜선을 뽑아도 그대로 작동합니다.

### 혹시 `index.html` 을 더블클릭했는데 화면이 비어 보이면
일부 브라우저는 `file://` 로컬 파일의 보안 정책이 엄격합니다. 그럴 땐 폴더에서
아래 한 줄을 실행하고 안내되는 주소(예: http://localhost:8000)를 열면 100% 확실합니다.

```bash
python3 -m http.server 8000       # 파이썬이 있으면
# 또는  npx serve .   /   php -S localhost:8000
```

## 🔑 로그인

- **각자 실제 계정 비밀번호**가 그대로 통합니다. (서버와 동일한 sha256 검증을 로컬에서 수행)
- 권한도 동일: 관리자(hansy)는 이름 변경·마크 등 편집 기능까지, 에디터/뷰어도 원래대로.
- 뷰어 전용 진입: `0715`

즐겨찾기·내 소재 토글, 소재 이름/서비스 편집 같은 편집 기능도 **그 자리에서 동작**하고,
브라우저에 저장되어 새로고침해도 유지됩니다. (오프라인이라 서버 대신 로컬에 기록됩니다.)

## 📦 폴더 구성

| 파일/폴더 | 내용 |
|---|---|
| `index.html` | 오프라인용으로 패치된 사이트 본체 |
| `data.js` | Supabase 스냅샷 전체 — 주간 성과 19,377행 · kv(이름/OCR/유튜브 제목 등) · 계정 · 마크 |
| `offline_shim.js` | 모든 Supabase RPC를 로컬 데이터로 대체하는 어댑터 |
| `sha256.js` | 로그인 비밀번호 검증(서버와 동일 알고리즘) |
| `fonts/` | Pretendard 폰트(로컬) |
| `assets/` | 미리보기 이미지·영상·YouTube 썸네일 (약 1,525개) + `manifest.js` |
| `favicon.svg` | 파비콘 |
| `fetch_assets.py`, `assets_manifest.tsv` | (라이트판 전용) 미리보기 자산 다운로드 스크립트 |

## 🖼️ 미리보기에 대해

- **Moloco 이미지·영상, Google 이미지 소재** → 로컬 파일로 그대로 재생/표시됩니다.
- **YouTube 영상 소재** → 유튜브는 스트리밍이라 오프라인 저장이 원천적으로 불가능해서,
  대신 **썸네일(정지 포스터) + ▶ 표시**로 보여줍니다. (지표·점수·분석은 전부 정상)
- **Meta 소재** → 원래도 "광고관리자에서 보기(로그인 필요)" 링크라, 온라인과 똑같이 카드로 표시됩니다.

---

## ⚠️ 라이트판을 받으셨다면 (미디어 미포함, 용량 ~15MB)

미디어 파일까지 담으면 용량이 커서(~1.1GB) 전송이 어려운 경우, 코드+데이터만 담은
라이트판을 드립니다. **인터넷 되는 곳에서 딱 한 번** 아래를 실행하면 미디어를 채워 넣습니다.

```bash
python3 fetch_assets.py
```

끝나면(약 1,525개 다운로드) 이후로는 인터넷 없이 완전 오프라인으로 동작합니다.
중간에 끊겨도 다시 실행하면 이어받습니다.
