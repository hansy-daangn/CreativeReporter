# CreativeReporter

퍼포먼스 소재 성과분석 대시보드입니다.<br/>
CSV를 화면에 끌어다 놓으면 즉시 분석됩니다.<br/><br/>
https://hansy-daangn.github.io/CreativeReporter/

## 지원 리포트

- **Moloco 크리에이티브 리포트** — `creative_preview`·`creative_type` 컬럼 자동 감지, 소재 단위 분석 + 이미지/영상 미리보기
- **Google Ads 자산 리포트** — 확장 소재 URL 미리보기, ROAS·전환 지표
- **ad_name / creative_group_name / adset_name** 기반 일반 CSV
