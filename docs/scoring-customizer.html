커스텀 채점 · 지표 비중 편집기 UI 시안 3종

각 매체가 어떤 지표를 어느 비중으로 채점할지 직접 고르고 조절하는 편집기입니다. 지표를 체크해 추가하거나 해제해 빼고, 네모 블록의 너비를 드래그하거나 숫자를 입력해 비중을 바꾸면 오른쪽 미리보기 순위가 즉시 다시 매겨집니다. 아래 세 가지 인터랙션 시안을 눌러 비교해 보세요.

이 페이지는 UI 시안(프로토타입)입니다. 표시되는 순위는 각 시안의 조작을 체감할 수 있도록 가상의 샘플 소재 10개를 실제 채점 로직(지표별 광고비 가중 백분위 → 비중 가중합 → 비교군 내 재배치)의 축약판으로 매긴 것입니다. 저장 버튼은 데모용이라 실제로 저장되지 않습니다. 기본값은 현재 사이트의 매체별 가중치(효율 85% + 규모 15%)를 그대로 재현합니다 — 세 시안 모두 규모(데려온 유저)를 다른 지표와 동등한 하나의 조절 대상으로 통합해, 앞으로 새 기준을 자유롭게 추가/조정하는 방향을 함께 제안합니다.
